# markdown-converter
A tool for converting markdown to pdf and word with specific template.

## Basic Usage
```shell
# Choose mdk.exe for windows, mdk for Linux

# Show help document
mdk help

# Convert sample.md to sample.docx
mdk word sample.md
```
